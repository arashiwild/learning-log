# npm vs yarn vs pnpm

- **npm**: package manager default Node.js
- **yarn**: lebih cepat (versi awal), punya yarn.lock
- **pnpm**: menggunakan symlink ke global store, hemat disk space dan instalasi lebih cepat untuk monorepo

Semua kompatibel dengan registry npm, perbedaan utama di performa dan cara penyimpanan dependency.
