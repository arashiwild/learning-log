# Git Basics

Perintah dasar Git:

```bash
git init                 # inisialisasi repo
git status               # cek status perubahan
git add <file>           # stage perubahan
git commit -m "message"  # simpan perubahan
git push origin main     # kirim ke remote
git pull origin main     # ambil perubahan terbaru
git log --oneline        # lihat history commit singkat
```
