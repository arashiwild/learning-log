# CSS Flexbox Dasar

Flexbox digunakan untuk layout 1 dimensi (baris atau kolom).

```css
.container {
  display: flex;
  justify-content: center;
  align-items: center;
  flex-direction: row;
}
```

- justify-content: mengatur posisi horizontal
- align-items: mengatur posisi vertikal
- flex-wrap: mengatur apakah item bisa wrap ke baris baru
