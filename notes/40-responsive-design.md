# Responsive Design

Membuat layout menyesuaikan ukuran layar menggunakan:
- Media queries (`@media (max-width: 768px) {...}`)
- Unit relatif (%, rem, vw/vh) daripada px tetap
- Flexbox/Grid untuk layout fleksibel
- Mobile-first approach: desain dari layar kecil dulu, lalu tambah breakpoint untuk layar besar
