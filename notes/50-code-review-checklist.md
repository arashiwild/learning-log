# Code Review Checklist

- Apakah kode mudah dibaca dan dipahami?
- Apakah ada test untuk perubahan ini?
- Apakah ada edge case yang terlewat?
- Apakah ada duplikasi kode yang bisa diekstrak?
- Apakah penamaan variabel/fungsi jelas?
- Apakah ada masalah keamanan (input validation, secrets)?
- Apakah perubahan ini sesuai dengan scope PR (tidak terlalu besar)?
